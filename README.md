# zadanie1

Installation:

* Make sure you have Apache, PHP, MySQL installed.
* Clone this repo to a folder on your server.
* Activate mod_rewrite, route all traffic to application to pub/ folder.
* Edit app/etc/config.php and set your database credentials.
* Execute the SQL statements from the install directory to setup database.

Go to project folder and run the composer install command:

`composer install`

Go to project folder and run the bower install command:

`bower install`
